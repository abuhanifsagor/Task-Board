Question-1: What are the different ways to select an element in the DOM?
Answer:

There are many different ways to select an Element in the DOM.

  1. By Tag - document.getElementsByTagName('tag-name')
  2. By Class - document.getElementsByClassName('class-name')
  3. By ID - document.getElementById('id-name')
  4. By Query Selector - document.querySelector('.class-name')
  5. By Query Selector All - document.querySelectorAll('.class-name')

Event delegation is a way of handling events by using the parent element and add an eventListener to it.

Answer:

Event bubbling is a method in the DOM where if a target element event is triggered, then its will go upwads like a bubble in the DOM tree. LIKE CHILD > PARENT > GRAND PARENT.
and its will tigger the all events from the tiggered point to the root element. Note: we can use event.stopPropagation() to stop the event from bubbling!

Question-5: How do you create, add, and remove elements using JavaScript?
Answer:

TO CREATE, ADD, AND REMOVE ELEMENTS USING JAVASCRIPT EXP:

  • createElement: document.createElement('tag-name')
  • appendChild: parentElement.appendChild(childElement)
  • removeChild: parentElement.removeChild(childElement)
  • remove: parentElement.remove(childElement)
